#מלך הפעלים 2.0 (The Verb King 2.0)

אפליקציית Web לימודית אינטראקטיבית המיועדת לתלמידי כיתות ב׳–ג׳ לתרגול הטיות פעלים בבניין קל (גזרת השלמים). האפליקציה פותחה כיישום SPA (Single Page Application) רספונסיבי המבוסס על HTML5, Tailwind CSS ו-JavaScript.

## 🚀 מאפיינים מרכזיים

* **ניהול מצב מרכזי (State Management):** כל הנתונים (משתמש, שלבים, תשובות, ניקוד) מנוהלים באובייקט אחד לסנכרון מלא.
* **מטריצת חידון דינמית:** 4 שלבי תרגול עם כיסוי מלא של כינויי הגוף (יחיד, רבים, מעורב).
* **מערכת ניתוח ביצועים (Analytics):** גרפים ויזואליים (Chart.js) ותובנות חכמות מבוססות ביצועים.
* **מעבדת תיקון (Correction Lab):** מנגנון ייחודי לתיקון טעויות בלבד הכולל מערכת רמזים פדגוגיים עם "קנס" ניקוד.
* **הפקת תעודת הצטיינות:** מסך סיום מעוצב להדפסה או צילום מסך.
* **עיצוב Mobile First:** ממשק מותאם למגע, כפתורים גדולים ונגישות לילדים.

## 🛠 טכנולוגיות

* **Frontend:** HTML5, CSS3 (Tailwind CSS CDN).
* **Logic:** Vanilla JavaScript (ES6+).
* **Charts:** Chart.js.
* **Fonts:** Heebo & Assistant (Google Fonts).

## 📂 מבנה הקובץ

הפרויקט בנוי בקובץ `index.html` יחיד המכיל 7 מסכים:
1.  **מסך כניסה:** קליטת שם פרטי ומשפחה.
2.  **מסך הנחיות:** יישור קו פדגוגי והסבר חוקי המשחק.
3.  **בחירת שורש:** בחירה דינמית מתוך מאגר שורשי בניין קל.
4.  **מטריצת החידון:** הליבה האינטראקטיבית (גוף x זמן).
5.  **אנליטיקה:** הצגת אחוזי הצלחה ותובנות AI.
6.  **מעבדת תיקון:** טיפול ממוקד בשגיאות.
7.  **תעודת הצטיינות:** סיכום חגיגי של ההישגים.

## 💻 התקנה והרצה

אין צורך בהתקנה מיוחדת או בשרת:
1.  שמור את הקוד שיוצר כקובץ `index.html`.
2.  פתח את הקובץ בכל דפדפן מודרני (Chrome, Edge, Safari).
